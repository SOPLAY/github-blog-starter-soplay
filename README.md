# Github-blog-starter-soplay (❗️개발 진행중)

## with Next.Js + TypeScript + Tailwindcss + Contentlayer

## Setting

1.`blog.config.js` 파일에서 해당 하는 부분 수정
`links`에서 해당 사항이 없는 부분은 **공백**으로 유지!

```js
module.exports = {
  title: '블로그 이름',
  author: '블로그 닉네임',
  description: '블로그 요약',
  url: '블로그 주소',
  links: {
    github: '깃허브 주소 (ex -> https://github.com/userNick)',
    linkedin: '',
    notion: '',
    kakao: '',
    discord: '',
    instagram: '',
  },
};
```
